# EduGrade Pro: Scratch 自動批改系統

本專案為基於 AI 的 Scratch 作業自動批改系統，專為資訊教師設計，用以輔助教學並大幅節省批改時間。本存放區提供完整的 Python 原始碼供交流與使用。

## 系統執行方式

本系統免安裝，需於 Google Colab 雲端環境執行，並搭配 Google Drive 讀取學生的作業檔案。

### 操作步驟

1. 從本 GitHub 存放區下載完整的程式碼檔案（例如 `.ipynb` 筆記本檔）。
2. 開啟 [Google Colab](https://colab.research.google.com/) 網站。
3. 點擊「檔案」>「上傳筆記本」，將剛剛下載的檔案上傳至 Colab。
4. 執行程式碼中的「步驟一」區塊，並依照畫面指示授權存取您的 Google Drive。
5. 執行「步驟二」主程式區塊。待程式啟動後，點擊底部輸出的網址以開啟網頁介面。
6. 於介面中輸入您的 API Key 即可開始全班批改。
